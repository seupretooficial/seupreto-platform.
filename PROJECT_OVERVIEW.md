### Project Overview: SeuPreto Platform

Plataforma própria para gerenciar a Barbearia Seu Preto e futuramente licenciá-la para outros barbeiros (SaaS).

#### Objetivos
- Agendamento online e cadastro de clientes.
- Histórico de cortes e controle financeiro.
- Painéis para Administrador, Barbeiro e Cliente.
- Notificações automáticas via WhatsApp/e-mail.

#### Tecnologias
- Visual Studio Code.
- Git/GitHub para versionamento.
- Arquitetura escalável para expansão SaaS.

#### Regras do Projeto
- Cada etapa deve ser concluída antes de passar para a próxima.
- Todo código precisa ser documentado.
- Foco inicial: Barbearia Seu Preto.
