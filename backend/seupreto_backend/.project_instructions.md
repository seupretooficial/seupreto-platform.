# SeuPreto Platform - Backend API

## Visão Geral
Sistema completo para gerenciamento de barbearia com objetivo de virar SaaS. Backend NestJS + TypeScript + Prisma + PostgreSQL.

## Arquitetura
- **Framework**: NestJS com TypeScript
- **ORM**: Prisma
- **Banco**: PostgreSQL (hospedado Abacus AI)
- **Autenticação**: JWT com 3 níveis (CLIENTE, BARBEIRO, ADMIN)
- **Documentação**: Swagger em /api-docs

## Estrutura de Módulos
- `PrismaModule` (global) - conexão com banco
- `AuthModule` - registro (/api/signup) e login (/api/auth/login) com JWT
- `ClientesModule` - CRUD de clientes (/clientes)
- `AgendamentosModule` - CRUD de agendamentos (/agendamentos) com validação de horários

## Modelo de Dados
- **user**: id, nome, email, senha (bcrypt), role (enum), telefone, ativo, timestamps
- **agendamento**: id, clienteId, barbeiroId, dataHora, servico, status (enum), observacoes, timestamps
- Relacionamentos: user -> agendamentos (como cliente e como barbeiro)

## Decisões Técnicas
- Soft delete para clientes (campo `ativo`)
- Validação de conflito de horário (intervalo de 1h entre agendamentos do mesmo barbeiro)
- Guards: JwtAuthGuard + RolesGuard para controle de acesso
- GlobalExceptionFilter para tratamento uniforme de erros
- Respostas em português

## Preparado para Expansão
- Histórico de cortes
- Controle financeiro
- Notificações
- Multi-tenant (SaaS)
